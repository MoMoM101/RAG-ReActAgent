# 社区行为准则

## 我们的承诺

我们希望 RAG Agent 社区保持开放、友善、专业和可协作。参与者不因年龄、身体状况、民族、性别认同与表达、经验水平、国籍、外貌、种族、宗教、性取向或技术选择而受到骚扰或歧视。

## 期望行为

- 对不同经验和观点保持尊重；
- 围绕事实、代码和可复现结果讨论问题；
- 给出建设性反馈，并接受合理修正；
- 尊重隐私、知识产权和安全披露流程；
- 在日志、截图和示例中主动删除敏感信息；
- 把社区和使用者的安全放在个人争论之前。

## 不可接受的行为

- 人身攻击、侮辱、威胁、骚扰或歧视；
- 发布他人的私人信息、凭据或未授权数据；
- 在被要求停止后继续进行不受欢迎的接触；
- 故意破坏讨论、Issue、Pull Request 或基础设施；
- 公开披露尚未修复的安全漏洞和利用细节；
- 其他在专业协作环境中明显不适当的行为。

## 适用范围

本准则适用于仓库、Issue、Pull Request、讨论区、发布页面以及代表本项目参与的其他公开空间。

## 执行方式

维护者可以编辑或删除不当内容，警告参与者，临时或永久限制其参与。处理时将考虑行为的严重程度、重复性、影响范围和修复意愿。

普通行为问题可以通过仓库维护渠道反馈；涉及安全漏洞或敏感信息时，请按照 [SECURITY.md](SECURITY.md) 私下报告。报告将尽量保密处理，但无法承诺绝对保密或固定处理时限。

## 维护者责任

维护者应一致、克制地执行本准则，说明必要措施的原因，并避免利用维护权限压制善意技术分歧。
